SendLite Fintech Hackathon

Overview

SendLite is a fintech hackathon project that explores a low-bandwidth digital remittance solution designed to make cross-border money transfers more accessible in areas with limited or unreliable internet connectivity.

The project focused on defining product requirements, documenting user needs, and designing a practical solution that balances usability, security, and accessibility.

My Role

I served as the Technical Business Analyst for the project. My responsibilities included:

* Defining product requirements
* Developing user stories and acceptance criteria
* Translating business needs into technical requirements
* Contributing to functional documentation
* Collaborating with teammates to refine the proposed solution
